Manjaro 安装配置

由于经常用到linux系统，折腾起来也蛮费劲的，想着以后万一换电脑了又得捣鼓一天，索性在这记录一下manjaro安装流程，网上当然也有，但由于版本问题往往要找很多教程才能成功弄出一个来，大多因为依赖兼容或者不够详细省略很多关键步骤。

ISO下载与制作启动盘

首先到下面网址下载好kde版本的iso，电脑下载rufus制作启动盘 

https://manjaro.org/get-manjaro/

安装过程：略（唯一要注意的就是分区，自行百度）

配置

任何一个linux装好系统后第一件事就是换源，不然下载速度爆炸 在终端执行以下命令 

sudo pacman-mirrors -i -c China -m rank
测试各个镜像源下载速度，选择延迟最低的一个 然后更新缓存，更新库 
sudo pacman -Syy
sudo pacman -Syu

安装vim（这个不用多说）

安装中文输入法

sudo pacman -S fcitx-im
sudo pacman -S fcitx-contigtool
sudo pacman -S fcitx-googlepinyin
修改配置文件开机自启： 
vim ~/exprofile
输入一下内容： 
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"
重启

安装浏览器edge

edge现在非常好用，几乎是离不开了，特别是阅读pdf

sudo pacman -S yay
yay -S base-devel
yay -S microsoft-edge-dev-bin
先安装yay包管理器，yay安装软件不用sudo，然后安装一大堆打包工具，因为后面有些包要我们自己编译 既然如此顺便把cmake和go也安装了吧，后面编译vim插件要用到。 
yay -S cmake
yay -S go

SHELL

manjaro自带zsh，我们还需要下载ohmyzsh来美化一下，但是现在ohmyzsh现在大概率因为不可说的原因无法访问 

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
curl: (7) Failed to connect to raw.githubusercontent.com port 443: Connection refused
这个时候我们可以去github下载项目，然后跑脚本（不是） 使用一下工具查看 raw.githubusercontent.com 的IP地址 https://site.ip138.com/raw.githubusercontent.com/ ，然后修改hosts文件 
sudo vim /etc/hosts
添加
查询到的国内可用ip raw.githubusercontent.com
修改完成后在运行 安装插件 语法高亮 
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
自动补全 
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
其他的什么一键解压跳转目录自己在找 修改.zshrc文件，添加插件，最后source ~/.zshrc成功 
plugins=(git
        zsh-autosuggestions
        zsh-syntax-highlighting
）

配置vim

首先下载vundle插件 

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
下载最难搞定的ycm插件，必需（git，cmake，go），没有要先下载。 
cd ~/.vim/vundle
git clone https://github.com/Valloric/YouCompleteMe.git
然后进入到YouCompleteMe目录下 
cd ~/.vim/bundle/YouCompleteMe
git submodule update --init --recursive
./install.py --all
等待编译完成后，直接把我这个配置copy过去
